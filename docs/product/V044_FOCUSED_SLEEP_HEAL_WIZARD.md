# v044 Focused Sleep HEAL Wizard

## Scope

This release is limited to `#/shine/sleep`. Sections 1–5 and every other SHINE, HEAL, and DR page remain unchanged.

## Pattern change

The Sleep page now ends with the guided HEAL Wizard only. The duplicate manual Action Path card and full task list were removed from this page.

Users who prefer manual exploration can continue to search and select the same lifestyle actions from the HEAL area. The underlying task library remains shared, so actions selected through HEAL, the wizard, Your Plan, and SHINE Path remain synchronized.

## Wizard flow

- Health considerations
- Sleep and work schedule
- Sleep experiences
- Daily-life factors
- Categorized matched actions

Selections are optional, multi-select, local-only, and are not diagnoses.

## Safety

- No dose, medicine change, diagnosis, or treatment instruction is generated.
- Severe sleepiness remains linked to existing safety wording.
- Clinical questions and tests remain available through connected HEAL and DR pages.
- Recommendation wording remains educational and conditional.

## Persistence

Wizard selections are stored in `profile.sleepWizard`. Selected actions remain in `profile.planTasks`.
