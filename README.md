# Patient Care v045 — Clarity Pass

Patient Care is a dependency-free, offline patient-education and personal planning application.

## v045 changes

- Full researched teaching is retained and shown through progressive disclosure.
- Existing warning-oriented teaching remains visible in an Important notice.
- Action Path groups use compact native disclosure controls.
- Placeholder content is excluded from ordinary lists, search, and related links.
- Exercise remains visible in SHINE as a non-clickable Coming soon card.
- Plan routes and task-core counts are corrected.
- The generic duplicate doctor-question control is removed.
- JSON backup is available once from Plan Overview.

No medical claims, editorial sources, clinical-review states, dosing locks, profile version, or saved-data keys were changed.

## Run locally

Use `open_local_preview.cmd`, or serve this directory with any local static HTTP server and open `index.html`.
